클라우드 컴퓨팅(Drivers for Cloud Computing) - 서비스적인 기능
- 인터넷으로 가상화된 IT 리소스를 제공하는 것을 의미합니다. 
  클라우드 서비스 플랫폼을 통해 종량제 결제 방식으로 제공되는 온디맨드 서비스형 컴퓨팅입니다.

	[ 특징 ]
  서비스 제공
  사용자가 직접 설정 가능해야 한다.
  아무대서나 접속 가능
  빠른 생성 가능(미리 만들고, 스냅샷을 통해 만듬)
  측정 가능(과금을 위해)
  (분산 컴퓨터, 가상화, 측정, 마음대로 디자인 가능)
	[ 이점 ]
  IT 비용을 줄임
  기업에 반영이 빠름
  확장 축소가 간편함

	[ 클라우드(서비스) 모델 ]
  IaaS : Infrastructure-as-a-Service : 인프라(네트워크, 컴퓨터, 스토리지(하드웨어))를 서비스처럼 제공
  PaaS : Platform-as-a-Service : 플랫폼(하드웨어(를 제어하는 기반까지), OS, DB, WAS)를 서비스처럼 제공
  SaaS : Software-as-a-Service : 전체 다 서비스(네이버 클라우드, 다음 클라우드)

	[ 필요 이유 ]
- 회사가 원한다. (관리면에서 이득, 성능은 좀 떨어짐)

	
	[ 서비스하는(배포하는) 모델 ]
  Public : 가상화를 공개적으로 가상화 제공 받음
  Private : 상대편 기업에 구축 해줌 - 운영은 상대편이 (우리 회사를 위한 클라우드 구축)
  Community : 다른 클라우드와 연계
  hybrid : Public + Private : 자기 기업을 위해 구축했지만, 사용자에게도 제공
  EX) 네이버 클라우드 : Public, SaaS


클라우드 슈(set) : 오픈 스택, VM(private), 아마존(public), 애져
